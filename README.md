# 🛡️ DocGuard (文档守卫)

> 让法律文书脱敏像魔法一样简单，且排版一丝不苟。

**DocGuard** 是一款专为法律、财务及办公场景打造的**原生文档脱敏工具**。它不仅能通过大语言模型 (LLM) 智能找出隐私实体，还能在**完全保留原文件排版、字体、样式和印章**的前提下，对原生 Word、PDF、Excel、PPT 和 Markdown 文件进行精准的脱敏与打码！

---

## ✨ 核心特性

- 🎯 **原生样式零破坏 (NER + Redaction)**：
  摒弃了传统的“让大模型重写整段话”导致的排版灾难。DocGuard 仅让大模型扮演“找茬员”（实体识别 NER），随后使用 Python 库在底层对文件进行精确的“坐标遮挡 (PDF)”或“片段替换 (Word/PPT)”。加粗、字号、表格、甚至**红头印章**均完美保留！
- 🔓 **灵活兼容任何大模型**：
  基于标准的 OpenAI API 接口协议通讯。既支持完全离线的本地部署（Ollama、LM Studio），实现敏感案卷**零泄漏、不触网**；也支持直接接入云端大模型（DeepSeek、通义千问、OpenAI）。
- 📦 **五合一全能格式支持**：
  - 📄 **Word (`.docx`)**：精准替换 Run 级别文本，复杂版式丝毫不乱。
  - 📕 **PDF (`.pdf`)**：基于几何坐标的纯黑遮挡 (Redaction)，印章与扫描图像完美存活。（*注：目前需 PDF 带有可复制的底层文本*）
  - 📊 **Excel (`.xlsx`)**：工作表与图表无损保留，精准替换单元格。
  - 📽️ **PPT (`.pptx`)**：支持幻灯片内的文本框、形状智能替换。
  - 📝 **纯文本 (`.txt` / `.md`)**：轻量级字符替换。
- 📝 **独家双轨输出机制**：
  文件处理完成后，除了生成一份“完美保留原排版”的格式文件（用于人类阅读与交付）外，还会**自动附赠一份同名的纯文本 `.md` 文件**。不仅能剥离冗余排版，还能极大地节省 Token，专供下游大模型（如 ChatGPT/DeepSeek）做进一步的案情总结与财报分析。
- ⚡ **秒级增量缓存与拦截网**：
  - **指纹缓存**：自动记录已脱敏的文件指纹，重复文件秒级跳过，绝不白白浪费您的 API Token 和时间。
  - **质量兜底校验**：内置防御性正则检测引擎，自动扫描最终文本。若发现疑似漏提的百家姓或法律称谓，会在终端发出高亮警告，辅助人工安全复核。
- 🚀 **一键开箱即用**：
  专为非技术人员设计的“沙盒机制”。双击 `.command` 脚本自动安装隔离的虚拟环境，**绝不污染你的 Mac/PC 系统全局配置**。

## 🛠️ 快速开始

### 1. 前置准备
1. 确保已安装 Python 环境。
2. 准备一个 **大语言模型 (LLM) API 服务**：
   - **本地离线方案 (推荐，保护隐私)**：你可以使用 [LM Studio](https://lmstudio.ai/)、[Ollama](https://ollama.com/) 或 vLLM 在本地运行模型。
   - **云端在线方案**：如果文件不涉密，也可以直接使用 DeepSeek、通义千问、OpenAI 等云端 API。
3. **关于大模型的选择**：
   - **最低要求**：模型需要具备良好的“指令遵循 (Instruction Following)”能力，能够稳定输出 JSON 格式数据，并且中文命名实体识别 (NER) 能力达标。一般建议选择 **7B 规模及以上** 的参数量。
   - **推荐模型**：强烈推荐使用 **Qwen 2.5 (通义千问) 7B 或 14B** 系列模型，它在零样本实体抽取上的表现极其稳定且精准。

### 2. 运行工具
1. 克隆或下载本仓库代码。
2. 进入 `docguard` 文件夹。
3. **Mac 用户**：直接双击运行 `一键DocGuard脱敏.command`。
4. **Windows 用户**：直接双击运行 `一键DocGuard脱敏.bat`。
   *(首次运行会自动在后台创建 `.venv` 虚拟环境并下载依赖，请耐心等待几分钟)*。
5. 运行后，程序会自动生成 `input` 和 `output` 文件夹。

### 3. 开始脱敏
1. 将需要脱敏的文件拖入 `input` 文件夹（红区）。
2. 在弹出的终端窗口中按回车键确认。
3. 去泡杯咖啡，DocGuard 会自动对文件进行解析、实体提取和精准替换。
4. 处理完毕后，脱敏好的文件将安静地躺在 `output` 文件夹（绿区）中。

## ⚙️ 进阶配置

程序通过读取环境变量或直接修改代码来配置你的大模型后端。你可以用任意文本编辑器打开 **`ner_engine.py`** 顶部，修改以下默认值：

- `DOCGUARD_API_BASE` (默认: `http://localhost:1234/v1`，这是 LM Studio 的默认地址。若用 Ollama 通常是 `http://localhost:11434/v1`，或填入云端 API 地址)
- `DOCGUARD_API_KEY` (默认: `lm-studio`。若使用真实的云端 API 请填入真实 Key)
- `DOCGUARD_MODEL` (默认: `qwen2.5-14b-instruct-1m`。请填入你实际运行的模型名称)

## ⚠️ 局限性与注意事项
- 纯图片扫描件 PDF 目前暂不支持直接画框。如果你手头是纯图片扫描件，请先使用 Adobe Acrobat 或 WPS 等软件对其执行一次“OCR 识别文本”（存为双层可搜索 PDF），然后再喂给 DocGuard 即可完美遮挡。
- 脱敏是一项严谨的工作，大模型难免会有漏网之鱼，**交付给客户前请务必进行人工复核检查！**
